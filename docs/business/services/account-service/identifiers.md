# Account Identifiers

Bank accounts need identifiers that can be used by customers, systems, and other banking operations.

## accountNumber

Internal account number generated by `account-service`.

Current format:

```txt
ACC-YYYY-NNNNNN
```

Example:

```txt
ACC-2026-000001
```

## CBU

Banking identifier generated from configured bank and branch values plus the account number.

The current CBU generator is simplified for the project. It is not a full official CBU implementation.

Current configurable values:

```txt
bank-code
default-branch-code
```

These values live in service configuration.

## Alias

Human-friendly identifier for the account.

Examples:

```txt
fede.bank.ars
mi.cuenta
ahorro-ars
```

Current rules:

```txt
- Lowercase letters.
- Numbers.
- Dots or hyphens.
- Between 2 and 6 segments.
- Maximum 80 characters.
- Unique when present.
```
