# Account Types

Account type describes the banking product represented by the account.

## Current Types

```txt
SAVINGS
CHECKING
SALARY
```

## SAVINGS

A basic savings account.

Common use:

```txt
- Receive transfers.
- Store money.
- Hold ARS or USD accounts.
```

## CHECKING

A checking account.

Common use:

```txt
- More advanced banking operation.
- Business or professional usage.
```

The current implementation does not support overdraft or checks.

## SALARY

A salary account.

Common use:

```txt
- Receive salary payments.
```

The current implementation does not model employers or salary agreements.
